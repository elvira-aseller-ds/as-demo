📦 Склад / Коробки


  • В модалку коробки добавлен таб с историей изменений коробки

    • История отображается от старых записей к новым, с правой стороны указано время

    • Типы действий, которые логгируются с никнеймом пользователя, который внёс изменения:

      • Смена тарифа

      • Установка/снятие метки "Сформировано"

      • Изменение даты отправки

      • Изменение комментария к коробке (от клиента / сторкипера)

    • Смена статуса коробки также логгируется, но без никнеймов

    • Рядом с меткой времени справа доступна кнопка копирования строки из истории

    • По кнопке "Копировать всю историю" - будет скопирована вся история коробки

    • История фиксируется для коробок с момента обновления, для старых коробок недоступна
